# Numpy-Basics
Basic NumPy practice programs and notes for learning arrays, reshaping, indexing, and mathematical operations used in AI/ML.

This repository contains my NumPy practice programs and notes while learning AI/ML fundamentals.

## Topics Covered
- Array Creation
- Shape, Size, Dimensions
- Reshape and Ravel
- Arange and Linspace
- Identity and Zero Arrays
- Axis Operations
- Sum, Argmin, Max
- Sorting
- Where and Count Non-Zero
- Array Addition
- Data Types

  

 Topics Covered
 1. Shape

Used to find the dimensions of an array.
 2. Size

Returns the total number of elements in the array.
 3. Arange

Creates arrays with a range of numbers.

 4. Empty

Creates an empty array with random existing memory values.

 5. Dtype

Shows the data type of array elements.
6. Zeros

Creates an array filled with zeros.
 7. Axis (0 and 1)

Used for operations across rows or columns.

* `axis=0` → column-wise
* `axis=1` → row-wise
8. Linspace

Creates equally spaced numbers between a range.
 9. Identity Matrix

Creates a square matrix with diagonal elements as 1.
 10. Array Creation from Python Structures

Converts lists or tuples into NumPy arrays.

 11. Reshape

Changes the shape of an array without changing data.

 12. Ravel

Converts multidimensional arrays into 1D arrays.

13. Sum

Finds the sum of array elements.

14.Argmin

Returns the index of the minimum value.
15. Max

Finds the maximum value in the array.

16. Ndim

Returns the number of dimensions of the array.

17. Nbytes

Shows total bytes consumed by array elements.
 18. Sort

Sorts array elements.
19. Where

Returns indices where a condition is true.

20. Count Non-Zero

Counts non-zero elements.

21. Array Addition

Performs element-wise addition.

 Conclusion

This repository helped me understand the fundamentals of NumPy including array manipulation, mathematical operations, reshaping, sorting, indexing, and data analysis basics used in AI/ML.
